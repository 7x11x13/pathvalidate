Installation
============

::

    pip install pathvalidate


Dependencies
============
Python 2.7+ or 3.3+
No external dependencies.


Test dependencies
-----------------
- `pytest <http://pytest.org/latest/>`__
- `pytest-runner <https://pypi.python.org/pypi/pytest-runner>`__
- `tox <https://testrun.org/tox/latest/>`__
